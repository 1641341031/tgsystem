## Windows下的安装：
- 下载nodejs并安装，最好使用v12.14.0版本，版本不对会导致npm install出错，下载地址：https://nodejs.org/dist/v12.14.0/node-v12.14.0-x64.msi
- 把项目clone到本地
- `npm install`，下载相关依赖
- `npm run serve`，运行项目
